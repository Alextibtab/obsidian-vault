# Topic / Title: Styling Shapes with Stroke and Fill

2023-01-07
04:05


# Notes
By default any shapes drawn will have a stroke of 1 pixel width and be filled in gray. To modify this behaviour there are the following functions:
- `stroke()`
- `noStroke()`
- `strokeWeight()`
- `fill()`
- `noFill()`
